Welcome to the Durian documentation.

The docs are organized into:

- **Getting Started** — installation and first run
- **Configuration** — Pkl files for accounts, rules, profiles, groups, keymaps
- **CLI Reference** — every `durian` subcommand
- **Keymaps** — vim-style navigation in the GUI

If you're looking for terse flag-level reference, the man pages are also installed:
`man durian-search`, `man durian-tag`, etc.

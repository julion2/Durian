Welcome to the Durian documentation.

The docs are organized into:

- **Getting Started** — install, configure your first account, send mail.
- **GUI** — the macOS app: compose, sidebar, search, drafts/outbox, attachments, notifications, keymaps.
- **CLI Reference** — every `durian` subcommand with examples.
- **Configuration** — typed Pkl files for accounts, rules, profiles, groups, keymaps.
- **Authentication** — OAuth (Gmail, Microsoft 365) or password.
- **Developers** — architecture and internals for contributors.

If you're looking for terse flag-level reference, the man pages are also installed:
`man durian-search`, `man durian-tag`, etc.

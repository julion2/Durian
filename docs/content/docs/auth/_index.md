Durian supports two authentication methods:

- **OAuth 2.0** — for Gmail and Microsoft 365. Tokens are stored in the OS keychain and auto-refreshed.
- **Password** — for everyone else (GMX, web.de, iCloud, Fastmail, custom IMAP servers). Passwords are stored in the OS keychain.

Pick the guide for your provider:

- [OAuth setup](oauth/) — Gmail, Microsoft 365
- [Password setup](password/) — everything else
